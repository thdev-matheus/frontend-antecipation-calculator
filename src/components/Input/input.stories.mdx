# Input

O componente de input, a nível de personalização é restrinto a apenas um atributo que é opcional, o width.
Caso não seja configurado, o input assumirá o width de 100% da tag pai qu o envolve.

Essa é uma forma de tornar ele flexível automaticamente.

Vejamos o exemplo do input com o width e do input se o width

<Canvas>
  <Story
    name="Input"
    args={{
      label: "label",
      name: "name",
      width: "20rem",
    }}
  >
    {(args) => <Input register={useFormUtils().register} {...args} />}
  </Story>
</Canvas>

## Atributos:

<ArgsTable of={Input} />

<br />

O input utilizado na aplicação é o que não possui o atributo width pois, em tdas as ocorrências ele está dentro de um componente form que limita o seu tamanho. dessa forma, mudando o tamanho do form, muda o tamanho do input deixando ele responsivo.

## Error Input

Caso queira exibir alguma mensagem de erro no input, basta passar uma string contend a mensagem de erro na propridade "error" do input

## Input Type

O Input também aceita os tipos relacionados a tag HTML input, embora na aplicação se use apenas o tipo "number"

Também foi retirado o botão de interação de click do input tipo "number".
