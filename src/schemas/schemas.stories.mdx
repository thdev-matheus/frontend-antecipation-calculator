# Um schema para validar um formulário...

<p>
  A aplicação utiliza a biblioteca yup para validação dos dados do formulário
  que por sua vez recolhe os dados necessários para a comunicação com a api.
</p>
<p>
  Dito isto, foi feito o schema que, nesta aplicação em particular, valida uma
  parte dos dados que vão ser enviados para a api.
</p>

```typescript
import * as yup from "yup";

export const schema = yup.object().shape({
  amount:
    yup /* deve ser um número, maior ou igual a 10 (esse valor será convertido para mandar para a api), obigatório */
      .number()
      .typeError("Digite um número válido")
      .required("Campo obrigratório")
      .min(10, "O valor mínimo é 10"),

  installments: yup /* deve ser um número, maior ou igual a 2, obigatório */
    .number()
    .typeError("Digite um número válido")
    .required("Campo obrigratório")
    .min(2, "O valor mínimo é 2"),

  mdr: yup /* deve ser um número, obigatório */
    .number()
    .typeError("Digite um número válido")
    .required("Campo obrigratório"),
});
```
