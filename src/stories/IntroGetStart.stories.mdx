## Executando a aplicação em contexto local

<br />

- Primeiro faça o clone parao seu repositório local com o comando:

```shell
$ git clone git@github.com:thdev-matheus/frontend-antecipation-calculator.git
```

ou utilizando a CLI do github:

```shell
$ gh repo clone thdev-matheus/frontend-antecipation-calculator
```

- Instale os pacotes necessários para o funcionamento da aplicação com:

```shell
$ npm i
```

- Para inicializar a aplicação execute:

```shell
$ npm run start
```

## Contribuindo com o deseenvolvimento da aplicação

- Se for contribuir com a aplicação, pode criar uma nova branch a partir da develop para criar seus blocos de código:

```shell
& git checkout -b <nome_da_branch>
```

- Sempre mantenha sua branch atualizada então antes de codar, garanta que vai estar tudo 100% pra não haver conflitos (faça isso sempre):

```shell
$ git pull origin develop
```

- Commits nunca não demais então para não perder o seu progresso esteja sempre commitando e fazendo pushes para a sua branch

```shell
$ git add .
$ git commit -m "mensagem do commit"
$ git push origin <nome_da_branch>
```

- Por falar em commits, por aqui eu sigo o padrão do conventional commits usando as palavras chave: "feat: mensagem", "fix: mensagem", "refactor: mensagem". Então vamos manter dessa forma porque assi é mais organizado.

- Seu trabalho está pronto? É hora de mandar um pull request. Faça um pull da develop para atualizar o código e depois faça um push para a sua branch. No github, solicite com um pull request a inserção do seu código na aplicação e não esqueça de comentar as mudanças e o motivo delas. Os comentários podem ser em português ou em inglês, você decide.

- Leia o código antes de codar e se atente ao padrão de código e de arquitetura de arquivos. Queremos com certeza evitar uma salada de hard codes por aqui.
